## [组合](https://leetcode-cn.com/problems/combinations/description/)

> 给定两个整数 `n` 和 `k`，返回范围 `[1, n]` 中所有可能的 `k` 个数的组合。

你可以按 **任何顺序** 返回答案。

**示例 1：**

```
输入：n = 4, k = 2
输出：
[
  [2,4],
  [3,4],
  [2,3],
  [1,2],
  [1,3],
  [1,4],
]
```

**示例 2：**

```
输入：n = 1, k = 1
输出：[[1]]
```

**提示：**

- `1 <= n <= 20`
- `1 <= k <= n`


[Discussion](https://leetcode-cn.com/problems/combinations/comments/) | [Solution](https://leetcode-cn.com/problems/combinations/solution/)


### 回溯 + 剪枝


```js
const combine_1 = (n, k) => {
  const res = [], ans = [];
  const backTracking = (n, k, i) => {
    // 终止条件 ans.length == k
    if (ans.length == k) {
      res.push([...ans])
      return;
    }
    // 剪枝
    const p = n - ans.length + 1
    for (let j = i; j <= p; j++) {
      ans.push(j); // 处理节点
      backTracking(n, k, j + 1); // 递归遍历
      ans.pop() // 回溯操作
    }
  }
  backTracking(n, k, 1)
  return res;
}
```
